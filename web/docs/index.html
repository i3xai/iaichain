1. 安装 iai

推荐用官网安装脚本，从 GitHub Release 拉取当前系统预编译包。

一键安装（推荐）

$ curl -fsSL https://iaiaiai.ai/install.sh | bash

指定版本 / 目录

# 指定版本
$ curl -fsSL https://iaiaiai.ai/install.sh | bash -s -- --version v0.4.2

# 安装到自定义目录
$ curl -fsSL https://iaiaiai.ai/install.sh | bash -s -- --dir ~/.local/bin

验证

$ iai version
# 期望类似：iai-chain 0.4.2
当前 Release 若只有 macos-aarch64 资产，其他平台请用源码构建：cargo build --release。完整说明见仓库 START.md / USAGE.md。

2. 启动：中继 + 队长 + 队员

本机开 3 个终端，模拟网络协调与两个独立节点。数据目录必须分开。

进程命令作用
中继iai relay --port 8790任务公告板、原子领取、入队状态（不记账）
队长IAI_HOME=~/.iai-cap IAI_RELAY=… iai serve --port 8787发任务、审批入队；控制台 :8787
队员IAI_HOME=~/.iai-mem IAI_RELAY=… iai serve --port 8788申请入队、领槽；控制台 :8788
 
# 终端 0 · 中继
$ iai relay --port 8790

# 终端 1 · 队长
$ IAI_HOME=~/.iai-cap IAI_RELAY=http://127.0.0.1:8790 iai serve --port 8787

# 终端 2 · 队员
$ IAI_HOME=~/.iai-mem IAI_RELAY=http://127.0.0.1:8790 iai serve --port 8788
角色控制台
队长http://127.0.0.1:8787/console
队员http://127.0.0.1:8788/console

首次启动会生成控制台密码（看日志），或：

$ IAI_HOME=~/.iai-cap iai password show
$ IAI_HOME=~/.iai-mem iai password show

3. 入队审批（领取前置）

未批准成员领取网络任务会得到 403。先完成申请 → 批准。

  1. 查队长 node id 
    $ IAI_HOME=~/.iai-cap iai node status
    记下类似 node.xxxx 的标识。
  2. 队员申请：打开队员控制台 → 左侧「团队」→「申请 / 审批」→ 填入队长 node id →「提交申请」。
  3. 队长批准：队长控制台同一面板 → 待审批列表 →「批准」。

4. 队长创建任务

创建「网络可领取」任务，才会出现在中继公告板供队员领取。

  1. 队长控制台 →「任务」→「+ 新建任务」。
  2. 填写标题与说明；仓库选开源 GitHub 地址，先点连通检测。
  3. 至少配置 1 个非队长角色，招募数 ≥ 1。
  4. 可见性选择 「网络可领取」（不要选「仅本机」）。奖金可先填 0。
  5. 点击「创建任务」，记下返回的 task id。

5. 队员领取任务

入队批准后，在队员控制台的网络任务列表领取开放槽。

  1. 队员控制台打开网络任务 / 可领列表（与任务视图相关区域）。
  2. 找到队长刚发布的任务与开放槽 → 点 「领取」。
  3. 成功后按钮变为「已领取」。状态为 claimed 即完成本手册 Phase 1 验收。
队员申请入队 → 队长批准 → 队长新建「网络可领取」任务 → 队员点领取
结果含义
已领取槽位 claimed，成功
403未加入该队长团队或尚未批准
409槽已被他人领走
未配置中继启动 serve 时未设置 IAI_RELAY

6. 常见问题

command not found: iai

检查 PATH，或重新执行安装脚本；源码构建可用 ./target/release/iai。

两个节点数据串了

确认 IAI_HOME 分别是 ~/.iai-cap 与 ~/.iai-mem，不要都用默认 ~/.iai。

申请入队失败

两边都要带 IAI_RELAY=http://127.0.0.1:8790，且中继进程已启动。

安装脚本 404

官网入口：https://iaiaiai.ai/install.sh。也可：curl -fsSL https://raw.githubusercontent.com/i3xai/iaichain/main/install.sh | bash。
